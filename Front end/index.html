# E-Commerce Platform - Frontend ## Project Overview A basic HTML/JavaScript frontend that interacts with the Flask backend to: - Display products. - Allow users to place orders. ## Tech Stack - **Frontend**: HTML, JavaScript (Fetch API) ## Setup Guide 1. Ensure the backend is running (see `backend/README.md`). 2. Open `index.html` in a browser (no server required). 3. Click "Buy Now" to place an order (mock transaction). ## Note - Uses `fetch` to call the backend API (CORS may require Flask configuration in production).
